# SIDTOOL

This is a tool to play with the SID registers.  Help file is embedded in the code when you press "?"

~~~
SID TOOL HELP MENU                     
-------------------                    
ARROW KEYS: MOVE FROM FIELD TO FIELD   
+/-:        ADD/SUB 1 FROM FIELD       
SHIFT +/-:  ADD/SUB 10 FROM FIELD      
CMD +/-:    ADD/SUB 100 FROM FIELD     
F1/F3/F5:   TOGGLE VOICE  1/2/3 ON/OFF  
T:          TOGGLE WAVEFORM TO TRIANGLE  
S:          TOGGLE WAVEFORM TO SAWTOOTH  
P:          TOGGLE WAVEFORM TO PULSE     
N:          TOGGLE WAVEFORM TO NOISE     
Y:          TOGGLE SYNC                  
R:          TOGGLE RINGMOD               
H/L/B:      TOGGLE HIGH/LOW/BAND FILTER  
F2/F4/F6:   TOGGLE FILTER VOICE 1/2/3    
M:          MUTE VOICE 3                 
V:          SAVE SETTINGS                
G:          LOAD SETTINGS                
1-9:        GET REGISTER SET             
SHIFT 1-9:  SAVE REGISTER SET            
?:          HELP                         
Q:          QUIT                         
~~~
